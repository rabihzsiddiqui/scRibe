# ambient audio assets

place a royalty-free lo-fi loop here as `lofi.mp3` to enable the lo-fi ambient sound option.

recommended sources:
- pixabay.com/music — search "lo-fi" (free, no attribution required)
- freesound.org — search "lofi loop", filter by CC0 license

the file should be:
- a short loop (30–120 seconds works well)
- instrumental only
- in mp3 format
- named exactly `lofi.mp3`

white noise and rain are generated procedurally via Web Audio API and require no files.
